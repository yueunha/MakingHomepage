HTML이란?

우리는 HTML을 이용해서 웹페이지 구조를 구축하고 내용들을 입력할거야.
HTML은 웹 사이트의 전체적인 구조를 잡아주고 내용을 넣을 수 있도록 만들어주는 프로그래밍 언어야.
Hypertext Markup Language을 줄여서 HTML이라고 부르지.

우리는 태그라는 걸 사용해서 HTML을 사용할 수 있어.
아래처럼 태그를 사용할 수 있는데, 이건 우리가 만들 사이트의 형식이야.

HTML안에 head와 body가 있어.
head에는 사이트의 이름이나 제목, body에는 사이트의 내용이 들어가지.
