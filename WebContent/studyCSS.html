CSS란?

css를 이용해서 웹페이지를 이쁘게 만들어 줄거야
css로 전반적인 웹 사이트의 스타일을 저장해둘 수 있어. 일종의 스타일 시트라고 할 수 있지.
HTML안에서도 어느정도 꾸밀 수는 있지만 css가 있다면
관련된 항목들에 관해서 스타일을 한번에 바꿔버릴 수 있으니까 아주 효과적이지?

css에 대해서도 직접 해보면 좋겠지만, 이번에는 내가 css는 만들어서 적용시켜줄거야.
아쉽지만 css에 관심이 있다면 여기를 클릭해서 공부해보는걸 추천할게!
css는 html과는 다르게 태그를 사용하지 않아. 대신 태그 이름을 가져와서 사용하지. 
        body {
          font-size: 14pt;
          text-align: center;
          }

        h1 {
          font-style: italic;
          font-weight: bold;
          text-decoration: underline;
        }
위의 내용을 한번 볼까?
body라는 태그에서 font-size로 글자 크기를 바꿔줬고, text-align으로 글자들이 가운데 오게 했네.
h1이라는 태그에서는 font-style을 italic으로 설정해서 기울어진 글자를 만들어줬어. 이렇게 기울어진거지!!
